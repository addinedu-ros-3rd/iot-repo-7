IoT 프로젝트 7조 저장소. 반려동물 자동 사료 배급기

## 반려동물 자동 사료 배급기 만들기 프로젝트

### function list

1. 일정시간이 되면 일정량의 사료 배급
	1. 즉시 배출 --> 지금 당장
	2. 8, 12, 24 --> 해당 시간 후에 배출 
2. 다음 사료 배급 시간까지 남은 시간을 디스플레이(배급기 전면부)에 출력
3. 음성 녹음 기능
	1. 사료가 일정량 없어지지 않으면 일정 시간 동안 음성 출력
	2. 처음 사료를 배급할 때 음성출력 
	3. 내가 원하는 목소리 추가/삭제/수정 가능
